This is the game centered, as web page content. If you move canvas to the top level of the body content, it will attempt to fill the screen, and surrounding it with other content, like banner ads or site information will not be possible.

Cocos2D doesn't clip as it should (panning screen 'overflows' game box). The clipping in this case is done by CSS.

This mode is friendly to integration with existing site layouts, or surrounding your game with links, ads, etc. on desktop browsers. The game can be any size, but preserve the square (or at least 'portrait') aspect ratio. It should fit down to 2:1, height:width. Obviously it probably won't work down to 'banana phone' aspects.
